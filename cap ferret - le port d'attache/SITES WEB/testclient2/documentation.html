PAGE LIME

pour l'édition via un navigateur web (firefox, etc...):

Connection au Compte "Page Lime"

Utilisez le formulaire de droite, avec l'identifiant et le mot de passe envoyés par SMS.

Vous disposez avec ces identifiants d'un compte "utilisateur", vous ne pourrez donc pas accéder à certains paramètres de configuration (gestion de la connection au serveur FTP, ajout/retrait de fonctions dans l'éditeur visuel..). Pas de risque ainsi d'erreurs critiques ou irréversibles.

Lien direct

Le Menu

Édition du Site: vous accéder à l'éditeur. Via ce bouton l'éditeur charge "par défaut" le site tel qu'il est sur le serveur (celui que voit le internaute). Il est conseillé dans un premier temps d'essayer plutôt avec les "brouillons".

Ouvrir le plan du Site: aucun intérêt pour vous, votre site est contenu dans une seule page.

Gestion des images: un gestionnaire d'image: télécharger facilement vos images, et modifiez-les. Vous pourrez ensuite les inclure dans le site, ou remplacer les images existantes par celles que vous avez vous-même téléchargé.

Gestions des Documents: un gestionnaire de documents (.pdf, .zip, etc...). Vous ne pouvez pas les afficher tel quel sur le site, mais seulement créer un lien vers ceux-ci. Pour cette démo, je n'ai pas activé la fonction de l'éditeur pour créer des liens.

Brouillons Récents: Un "brouillon" est une copie du site que vous pouvez sauvegarder sans la publier réellement. C'est une fonctionnalité essentielle à utiliser systématiquement, afin de pouvoir revenir à tout moment à une version antérieure!

L'éditeur de site

Vous voyez la version en ligne de la page. (Mais vous avez un brouillon en cours): cliquez ici pour basculer sur le brouillon, c'est plus sûr ;-).

Les Bulles 'edit-T" indique les parties ou blocs qui sont éditables. Cliquez simplement dessus, "et voilà"!

Actions des Pages:

Publication: mise en ligne du site tel que vous venez de l'éditer.

Sauvegarder le Brouillon: à chaque sauvegarde une nouvelle "version" est crée. La liste des versions s'affichera sur la gauche. Ainsi, si lors de la mise en ligne finale, des problèmes apparaissent, vous pouvez "revenir" en arrière, jusqu'à une "version" qui fonctionnait bien. Prenez de suite l'habitude d'utiliser les brouillons, par sécurité, mais aussi parce qu'il vous permettront de "comparer" différentes modifications de style ou de mise en page.

Les versions sont numérotées par exemple "2.3" : le chiffre avant le point (ici "2") s'incrémente à chaque mise en ligne (publication) réelle, le chiffre aprés le point (ici "3") s'incrémente à chaque sauvegarde d'un brouillon". Dans cet exemple il s'agit donc du troisième brouillon sauvegardé aprés la publication réelle

remplacer une image

1

Aprés avoir cliqué sur une bulle "edit-T", l'éditeur s'affiche. Faites un "click droit" sur l'image que vous souhaitez remplacer. Sélectionner "Insérer/éditer l'image"

 

2

Dans la fenêtre "Insérer/éditer une image" qui s'affiche, cliquez sur le bouton à droite du paramètre "URL de l'image"

Vous allez être redirigé sur le gestionnaire d'image. Cliquez simplement sur l'image pour la sélectionner. Cliquez sur "mettre à jour"

 

3

Vous n'avez (normalement) pas à vous soucier de la taille des images, le code HTML les redimensionne de façon "dynamique". Il est préférable quand même de conserver les mêmes ratios longueur/largeur pour la mise en page.

 

Pour cela, dans la fenêtre "Insérer/éditer une image", cliquez sur l'onglet "apparence" et notez les dimensions de l'image que vous souhaitez remplacer avant d'ouvrir le gestionnaire d'image.

Dans le gestionnaires d'image, survolez l'image que vous avez choisi pour faire apparaître le bouton "Optimiser"

Sélectionner ensuite "adjustment", puis "crop". Recadrez ensuite l'image au dimensions voulues

 

 

Voilà, n'hésitez pas à tester le gestionnaire, à vérifier que vos modifications s'affiche correctement aprés publication, sauvegardez un maximum de brouillon au début pour tout tester, vous pouvez accéder à l'historique via le bouton "outils des pages" en haut à droite.

Il s'agit juste d'une "copie test", l'original du site est sauvegardé.

Si vous souhaitez ne pas enregistrer vos modifications sur un brouillon ni les publier, cliquez simplement sur le bouton "fermer" en haut à droite, vous reviendrez à la page d'accueuil, puis au dernier brouillon enregistré.

Pour voir (ou re-charger la version en ligne, en cas de mauvaise manip', ou pour voir les dernières mises à jours que j'effectue de mon côté - et qui n'affecteront pas vos modifications) cliquez sur la flêche à droite de la barre d'adresse où s'affiche l'adresse du site

Le message relatif à Google Map au chargement de la page est normal, la carte fonctionne normalement sur le site en ligne.
